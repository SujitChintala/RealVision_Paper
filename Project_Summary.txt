RealVision Project - Accurate Implementation Details
=====================================================

DATASET:
--------
- Training Data: data/train/ (FAKE and REAL folders)
- Test Data: data/test/ (FAKE and REAL folders)
- Total Images: ~20,000 (balanced)
- Used for Training: 2,000 images (limited for fast prototyping)
- Train/Val Split: 70/30 (1,400 train, 600 validation)
- Test Set: 20,000 images

MODEL ARCHITECTURE:
------------------
- Backbone: ResNet-18 (pre-trained on ImageNet)
- Classification Head:
  * Linear(512, 512)
  * BatchNorm1d(512)
  * ReLU
  * Dropout(0.5)
  * Linear(512, 2)
- Total Parameters: ~11 Million
- Model Size: ~45 MB

TRAINING CONFIGURATION:
----------------------
- Framework: PyTorch 2.0+
- Optimizer: Adam
  * Learning Rate: 1e-4 (0.0001)
  * Weight Decay: 1e-4 (0.0001)
- Loss Function: CrossEntropyLoss
- Batch Size: 64
- Epochs: 10
- Learning Rate Scheduler: StepLR (step_size=7, gamma=0.1)
- Device: CPU (works most efficiently on GPU if available)

DATA PREPROCESSING:
------------------
- Input Size: 224×224 pixels
- Normalization: ImageNet statistics
  * Mean: [0.485, 0.456, 0.406]
  * Std: [0.229, 0.224, 0.225]

DATA AUGMENTATION (Training Only):
----------------------------------
- Random Horizontal Flip
- Random Rotation (±10 degrees)
- Color Jitter:
  * Brightness: ±20%
  * Contrast: ±20%
